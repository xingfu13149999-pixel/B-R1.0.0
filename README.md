# 语纪 · 后台管理（B 端）

管理员使用的后台管理系统，与平板端项目分离、独立仓库部署。

- **设计尺寸**: 1920×1080（Figma 1.0.首页）
- **技术栈**: Vue 3 + Vite + Element Plus + Pinia + Vue Router

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 路由

| 路径     | 说明     |
|----------|----------|
| /        | 首页     |
| /login   | 登录     |

## 与平板端关系

- **ai-admin**：平板端（工作台、客户、项目、访谈等），面向一线。
- **admin**：本仓库，B 端后台，面向管理员，管理数据/统计等。

两者共用同一后端 API，部署与代码完全独立。
